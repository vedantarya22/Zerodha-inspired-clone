** Zerodha Clone**

A full-stack stock trading platform clone inspired by Zerodha, built to simulate real-world trading dashboard functionality and secure user authentication.

**Features**

User registration and login with secure authentication

JWT-based session management

Interactive trading dashboard UI

Portfolio tracking and order management simulation

RESTful APIs for client-server communication

Responsive and modern UI design

** Tech Stack**

Frontend: React.js

Backend: Node.js, Express.js

Database: MongoDB

Authentication: JSON Web Tokens (JWT)

Architecture: REST API-based client-server model
